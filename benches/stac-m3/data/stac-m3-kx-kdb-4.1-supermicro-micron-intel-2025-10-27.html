STAC-M3 Benchmark Results: KX kdb+ 4.1 on Intel, Supermicro, Micron

Reports

Read the report

Detailed configuration information are now available to eligible members at the link above. To learn more about subscription options, please contact us.

STAC recently performed STAC-M3™ benchmarks on a stack featuring KX’s kdb+ 4.1 database system sharded across six Supermicro Storage SuperServer SSG-222B-NE3X24R servers. Each system contained 24x Micron 9550 MAX NVMe SSDs and 2x Intel Xeon 6767P 64-core CPUs. Unlike many previous tests using a shared disk array, this solution used local storage across all nodes. 

STAC-M3 is the industry standard enterprise tick-analytics benchmark for database and hardware stacks that manage large time series of market data. It provides a vendor-neutral framework for evaluating the performance, efficiency, and scalability of time-series databases used in financial analytics and backtesting. 

The benchmark suite includes two sets of workloads: Antuco, the baseline suite for performance under fixed conditions, and Kanaga, an optional scaling suite that tests responsiveness and throughput at larger data volumes and user loads. 

In these latest tests, the combination of Intel, Micron, Supermicro, and KX delivered notable performance. The system: 

  • Outperformed all publicly disclosed results on 19 of 24 Kanaga mean-time response benchmarks.
    STAC-M3.β1.100T.YR{1,2,3,4,5}VWAB-12D-HO.TIME
    STAC-M3.β1.1T.{2,3,4,5}YRHIBID.TIME
    STAC-M3.β1.100T.YR{1,2,3,4,5}-12D-HO.TIME
    STAC-M3.β1.50T.YR{1,2,3,4,5}VWAB-12D-HO.TIME

 

  • Outperformed all publicly disclosed results on 3 of 3 Antuco 50-user and 100-user
    1.36x faster than a DDN based SUT (KDB221014) on STAC-M3.β1.100T.STATS-UI.TIME
    2.12x faster than an Intel Optane based SUT (KDB210428) on STAC-M3.β1.100T.VWAB-12D-NO.TIME
    1.29x faster than an Intel Optane based SUT (KDB210428) on STAC-M3.β1.50T.STATS-UI.TIME

 

  • Outperformed all publicly disclosed results on 3 of 5 Kanaga throughput benchmarks.
    1.77x faster than a Pure Storage based SUT (KDB231122) on STAC-M3.β1.1T.3YRHIBID.MBPS
    2.70x faster than a Pure Storage based SUT (KDB231122) on STAC-M3.β1.1T.4YRHIBID.MBPS
    2.51x faster than a Pure Storage based SUT (KDB231122) on STAC-M3.β1.1T.5YRHIBID.MBPS

 

The Stack Under Test included: 

  • STAC Pack
    • STAC-M3 Pack for kdb+, Rev I[1]
  • Database
    • KDB 4.1 (2025.02.18) running in shard mode
  • Database nodes
    • 6x Supermicro Storage SuperServer SSG-222B-NE3X24R
    • 2x Intel® Xeon® 6767P @ 2.40GHz
    • 16x 128GB DDR5-6400 @ 6400MT/s [2 TB total]
    • NVIDIA® ConnectX-7 400Gb IPoIB
    • OS RHEL9.6 (Plow)
  • Storage system (direct attached / local to each node)
    • 24x 12.8 TB Micron™ 9550 MAX NVMe™ SSD per server
    • PCIe® Gen5
    • 144 SSDs in total, 23x6 were used for data (1.84PB)
  • Network
    • MQM9700-NS2F, NVIDIA® 64-Port NDR 400G InfiniBand Data Center Switch

 

The report provides detailed performance metrics, methodology, and visualizations for both Antuco and Kanaga test suites with additional configuration information and results available to STAC Insights subscribers.
